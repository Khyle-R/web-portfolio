Foodea

Description:

Foodea is the Thesis/Capstone project me and my groupmates made during college. Being the project manager, it was the first big project I handled and I learned a lot from it. Foodea is a food delivery application that uses machine learning tools to recommend foods based on the users preferences, activities within the app that also factors the calorie intake of the user. We implemented two A.I. models which is the collaborative and content-based system, we used reactNative for the user and rider application, laravel and bootstrap5 for the website, JawsDb for the database, AWS s3-buckets for the storage of data, Google Maps API for the tracking of delivery, and lastly heroku cloud for the deployment of the website and handling of APIs. Overall the project consists of a Website, a user application, delivery rider application and lastly a python program for the machine learning

FitTracker

Description:

FitTracker is an AI exercise companion that tracks the user's proper posture during a specific set of exercises it also count how many reps you make and plays sound cues if you accomplished the exercise properly. It uses OpenCv and pipes for the tracking of users movements and made using python.
